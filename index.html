innounp Inno Setup 解包工具 innounp The Inno Setup Unpacker

基于 innounp 0.50 （作者：ariman）
支持 Inno Setup 版本 2.0.7 至 6.7.1 Based on innounp version 0.50 by ariman
Supports Inno Setup versions 2.0.7 through 6.7.1

Inno Setup 是一款流行的软件安装程序制作工具。遗憾的是，官方并未提供解包工具——要从自解压可执行文件中提取文件，唯一的方法是运行安装程序。 innounp 是一个用 Delphi 编写的控制台程序，用于检查 Inno Setup 压缩包并提取选定的文件。 Inno Setup is a popular program for making software installations. Unfortunately, there is no official unpacker - the only method of getting the files out of the self-extracting executable is to run it. innounp is a console program written in Delphi to inspect Inno Setup archives and extract selected files.

源代码已适配当前 Delphi 版本（自 Delphi 2009 起，Unicode 字符串成为原生类型）。 The source code is adapted to the current Delphi versions (Unicode strings are the native type since Delphi 2009).

核心特性 Key Features

innounp 的独特优势 What makes innounp stand out

开源且基于 IS 源码 Open Source & IS-Based

innounp 是开源项目，基于 Inno Setup 源代码开发。因此，它更有可能支持未来的 IS 版本，保持与最新安装包格式的兼容性。 innounp is open source and based on IS source. Therefore, it is more likely to support future IS versions, maintaining compatibility with the latest installer formats.

恢复安装脚本 Script Recovery

能够恢复安装脚本（.iss 文件）的部分内容，包括注册表更改和编译后的 Innerfuse/RemObjects Pascal 脚本（如果可用）。 It recovers portions of the installation script (.iss file), including the registry changes and the compiled Innerfuse/RemObjects Pascal Script, if available.

使用方法 Usage

innounp 是一个控制台应用程序，使用命令行选项来确定要执行的操作。如需更友好的图形界面，推荐使用 Inno Setup Unpacker - Windows GUI。 也可集成到批处理脚本中实现自动化操作。 innounp is a console application, and it uses command-line options to find out what to do. For a more human-friendly interface the Inno Setup Unpacker - Windows GUI is recommended. Integration into batch scripts is possible for automation. 

innounp [command] [options] <setup.exe or setup.0> [@filelist] [filemask ...]

命令 Commands

(无) 显示安装包的一般信息 Display general information about the installation archive
-v 详细文件列表（含大小和时间戳） Verbose list of files (with sizes and timestamps)
-s 简短文件列表（不含大小和时间戳） Short list of files (without sizes and timestamps)
-x 从安装包中提取文件（到当前目录，另见 -d） Extract the files from the installation (to current directory, also see -d)
-e 提取文件不带路径 Extract files without paths
-t 测试文件完整性 Test files for integrity
-l 显示安装包支持的语言列表 Show list of languages supported by the installation
-i 显示所有支持的 Inno Setup 版本 Show list of all supported Inno Setup versions

选项 Options

-b 批处理（非交互）模式 Batch (non-interactive) mode
-q 提取时不显示进度 Do not indicate progress while extracting
-m 处理内部嵌入文件 Process internal embedded files
-pPASS 使用密码解密安装包 Decrypt the installation with a password
-dDIR 提取文件到指定目录 Extract the files into DIR
-cDIR 指定安装包内的当前目录 Specifies that DIR is the current directory in the installation
-n 不尝试解包新版本 Don't attempt to unpack new versions
-fFILE 从文件读取密码 Reads the password from FILE
-a 处理所有重复文件的副本 Process all copies of duplicate files
-y 对所有查询自动回答"是" Assume Yes on all queries
-o 禁用彩色控制台输出 No colored console output
-h 不显示程序信息标题 Do not display headline with program info
-w 生成不使用 UTF-8 的脚本 Generate install script without UTF-8 conversion
-r 使用 Windows 区域设置格式 Use Windows regional settings for separators and date/time format
-u 控制台输出使用 UTF-8 Use UTF-8 for console output instead of ANSI

退出代码 Exit Codes

0 无错误 No errors
1 版本不受支持 Version not supported
2 安装包损坏或不兼容 Corrupt or incompatible setup file
3 其他错误 Other error

如果安装包含 ..setup.0（未使用 SetupLdr 制作），请对 setup.0 而不是 setup.exe 运行 innounp。 If an installation has ..setup.0 (it is made without using SetupLdr), run innounp on setup.0 instead of setup.exe.

要提取特定目录中的所有文件，请使用 dirname\*.*，而不是仅使用 dirname。默认情况下，所有文件都提取到当前目录。使用 -d 可覆盖此行为。 To extract all files from a specific directory, use dirname\*.*, not just dirname. By default all files are extracted to the current directory. Use -d to override this behavior.
